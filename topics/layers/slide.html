@layer

czyli warstwy w css

kontrola kolejności

                @layer a, b, c;
zagnieżdzanie warstwy 

                    @layer reset.type, default.type, reset.media, default.media;

                    @layer defaults {
                        /* un-layered defaults (higher priority) */
                        :any-link { color: rebeccapurple; }

                        /* layered defaults (lower priority) */
                        @layer reset {
                          a[href] { color: blue; }
                        }
                    }
importowanie do warstwy 
            
            @import url('example.css') layer(foo);
warstwy anonimowe 
            
            @layer { /* ... */ }
